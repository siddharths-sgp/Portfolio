Embedded Systems

Facial Recognition Vending Machine

Final-year capstone project: a smart vending machine prototype using real-time facial recognition for secure access and product dispensing. MATLAB handled image capture and processing using a PCA-based algorithm — selected over Eigenfaces and 2D-MBPCA for its balance of accuracy and performance on constrained hardware — while a Raspberry Pi 4B managed embedded control, GPIO-driven actuators and transaction logging. A MATLAB–Apple Silicon compatibility issue was resolved mid-build using Rosetta 2, and the whole system was developed under a formal HSE risk assessment and university Research Ethics Approval.

  • Achieved over 70% recognition accuracy using a PCA-based algorithm
  • Integrated hardware control via GPIO outputs, including LED status indicators
  • Built a local, timestamped CSV data-logging system for full transaction traceability
  • Delivered a fully working physical prototype, not just a simulation
Duration Sept 2024 – Jan 2025Tools FlexSim, PLC Ladder Logic, HMI
Industrial Automation

Apple Sorting & Packaging Automation

Designed and simulated a multi-station fruit-sorting line — two varieties (Braeburn and Gala) move through washing, quality classification and palletising stages, modelled end-to-end in FlexSim to validate conveyor throughput and cycle time before any physical build. PLC ladder logic then drove the real control layer: conveyors, wash-cycle timers, sensor-triggered sorting and pallet-transfer actuators, with an HMI layered on top for manual override and live monitoring.

  • Built a functional model of a complete industrial sorting process, from raw input through to class-sorted, palletised output
  • Implemented timer-driven wash cycles and count-based pallet triggers using ladder logic function blocks (TON/TOF/CTU)
  • Integrated fault detection and emergency-stop logic wired into every rung of the control sequence
  • Deepened understanding of process control, throughput planning and automation flow
Duration Nov 2022Tools Arduino, Pneumatics, Relays, Sensors
Embedded Systems

Electro-Pneumatic Pick-and-Place System

A bench-built electro-pneumatic system for automated object handling, combining proximity-sensor detection with relay-driven pneumatic actuators. A cascade circuit approach sequenced the actuator logic without needing a dedicated microcontroller for timing, giving consistent, repeatable motion across a full pick, transfer and place cycle on a twin-conveyor rig.

  • Automated pick-and-place operation for lightweight parts across two conveyor stages
  • Demonstrated electro-pneumatic control using relays and basic sensing components
  • Improved understanding of actuator timing, cascade sequencing and signal processing
Duration Sept 2024 – Jan 2025Tools Unity, C#, NavMesh, VR Prototyping
Digital Twin

3D Interactive Factory Simulation

A digital-twin-style simulation of a factory floor, built in Unity across two linked environments — an interactive scene hub and a full manufacturing walkthrough modelled around basketball production, moving through dedicated manufacturing, packing and storage zones. Autonomous agents navigate the space using Unity's NavMesh system, with custom C# scripting handling pathfinding, zone transitions and interaction between dynamic objects, all built to a VR-ready standard for future headset deployment.

  • Created a VR-ready environment for interactive, first-person simulation
  • Implemented AI-based agent navigation, zone-to-zone transitions and a working UI
  • Modelled a complete manufacturing-to-storage flow to demonstrate digital twin concepts
